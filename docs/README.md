## 🌟 推荐阅读顺序

如果你是第一次使用 LinuxStudio，建议按以下顺序阅读：

1. **[快速安装指南（嵌入式）](QUICK_INSTALL_EMBEDDED.md)** - 快速上手
2. **[安装指南](INSTALLATION_GUIDE.md)** - 了解详细选项
3. **[调试指南](DEBUG_GUIDE.md)** - 遇到问题时查阅
4. **[开发者指南](DEVELOPER_GUIDE.md)** - 深入了解框架
